# 웰스토리 메뉴 보드 (Streamlit 버전)

삼성 부산 캠퍼스의 웰스토리 메뉴를 조회하고, 투표 및 댓글 기능을 제공하는 Streamlit 웹 애플리케이션입니다.

## 주요 기능

### 1. 🍽️ 오늘의 메뉴
- 날짜별 점심 메뉴 조회
- 메뉴 이미지, 칼로리, 구성 정보 표시
- 웰스토리 API에서 가져온 실제 평점 표시
- 메뉴별 좋아요/별로 투표
- 메뉴별 댓글 작성 및 조회

### 2. 📋 자유 게시판
- 자유롭게 글 작성
- 게시글별 댓글 기능
- 실시간 업데이트

### 3. 📊 통계
- 인기 메뉴 TOP 5
- 전체 투표 현황
- 메뉴별 좋아요율 분석

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
streamlit run welstory_app.py
```

3. 브라우저에서 자동으로 열립니다 (기본: http://localhost:8501)

## 사용 방법

### 초기 설정
1. 사이드바에서 웰스토리 아이디와 비밀번호 입력
2. "로그인" 버튼 클릭

### 메뉴 조회
1. "🍽️ 오늘의 메뉴" 페이지에서 날짜 선택
2. 메뉴 정보, 이미지, 평점 확인
3. 좋아요/별로 투표 가능
4. 댓글 작성 및 확인

### 게시판 이용
1. "📋 자유 게시판" 페이지 선택
2. "새 글 작성" 클릭하여 글 작성
3. 게시글 클릭하여 댓글 작성

### 통계 확인
1. "📊 통계" 페이지에서 인기 메뉴 확인
2. 전체 투표 현황 조회

## 데이터 저장

모든 데이터는 `data/` 디렉토리에 JSON 형식으로 저장됩니다:
- `votes.json`: 투표 데이터
- `comments.json`: 메뉴 댓글 데이터
- `board.json`: 게시판 글 데이터

## 주의사항

- 웰스토리 API 로그인이 필요합니다
- 코드에서 `REST000595` (삼성 부산 전기)로 설정되어 있습니다
- 다른 식당을 사용하려면 `restaurantCode` 값을 변경하세요

## 기존 Flask 버전과의 차이점

### 제거된 기능
- Mattermost 웹훅 전송 기능
- 스케줄러 자동 실행 기능

### 추가된 기능
- 웹 인터페이스를 통한 메뉴 조회
- 메뉴별 투표 시스템
- 메뉴별 댓글 기능
- 자유 게시판
- 통계 대시보드
- 날짜별 메뉴 조회

## 커스터마이징

### 식당 변경
`welstory_app.py`의 `get_menu()` 함수에서 `restaurantCode` 값을 변경:
```python
params = {
    "restaurantCode": "YOUR_RESTAURANT_CODE",
    ...
}
```

### 스타일 변경
파일 상단의 CSS 섹션을 수정하여 디자인 변경 가능

### 기능 추가
- `load_votes()`, `save_votes()` 패턴을 따라 새로운 데이터 타입 추가 가능
- 새로운 페이지는 `show_*_page()` 함수 형태로 추가

## 라이선스
